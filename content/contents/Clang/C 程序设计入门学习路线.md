## 程序设计与 C 语言

初始的 Unix 操作系统是使用汇编语言开发的，由于汇编语言的特性导致这个版本的 Unix 无法在其他计算机中使用，因此只能重写操作系统。并且当时的高级语言是无法满足开发需求的，因此 Ken Thompson 在 BCPL 语言的基础上发明了 B 语言。

三年后，Dennis Ritchie 和 Brian Kernighan 又在 B 语言的基础上重新设计了一门计算机语言，称之为 C 语言。在此之后，整个 Unix 便采用 C 语言进行重写美，国国家标准协会（ANSI）也正式将 C 语言标准化。

### 计算机与编程语言

编程语言是专用于计算机，能够使程序员比机器语言更加准确地表达其目的的工具，目前编程语言已超过上千种。

### C 语言

C 语言经过长时间的发展，到现在也能长盛不衰，主要的原因是它具有其他语言所不具备的鲜明特点。

1. 低级语言

所谓低级语言，甚至 C 语言能够直接和操作系统、内存，甚至和硬件对话，更加接近系统底层。该特性能够实现程序和硬件的交互，以及满足对性能要求极高的程序。

2. 可移植性

文章开头提到，早期的 Unix 由于使用汇编语言开发导致系统不能够在其它设备运行，但是使用 C 语言重构的 Unix 系统能够移植到其他的计算机架构。相比汇编语言，C 程序可以相对简单的一直到各种硬件架构和操作系统。

3. 简单性

C 语言的语法相对简单，语法规则不算太多，也几乎没有语法糖。换句话说，如果存在两种语法能够完成一件几乎相同的事情，C 语言就只会提供一种语法，大大减少了语言的复杂性。并且，C 语言都是基础语法，复杂的数据结构都需要自己构造。

4. 灵活性

> 信任程序员，不要妨碍他们做事。

C 语言对程序员的限制很少，不提供内存自动清理功能，让程序员自己管理内存，也不提供类型检查、数组的负索引检查、指针位置的检查等保护措施。虽然在这方面舍弃了一定的安全性和 Debug 的方便，凡是对于高级程序员来说却有更大的编程自由。
